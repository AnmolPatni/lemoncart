# 🍋 LemonCart

LemonCart is a responsive e-commerce storefront developed using HTML5 and CSS3 as part of the Summer Internship Major Project.

## Project Description

LemonCart is a modern online shopping website that showcases various fashion and lifestyle products including clothing, shoes, bags, watches, beauty products, perfumes, and accessories.

The project is designed to provide a clean and user-friendly shopping experience with a premium yellow and white theme.

## Technologies Used

- HTML5
- CSS3
- Responsive Web Design

## Features

- Responsive Homepage
- Product Listing Page
- Product Details Page
- Shopping Cart
- User Login
- User Registration
- Checkout Page
- Contact Page
- Modern UI Design
- Fully Offline Compatible

## How to Run

1. Download or clone the project.
2. Open the LemonCart folder.
3. Double-click `index.html` or open it using any web browser.
4. No internet connection is required.
